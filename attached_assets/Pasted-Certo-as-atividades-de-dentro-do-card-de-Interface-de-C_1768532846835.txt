Lembre-se que para as atividades estarem verdadeideramente construidas, elas precisam serem construidas através do modal de Editar atividades, para que todo o conteúdo de dentro de cada campo de dentro do modal, seja util para a criação do conteúdo da atividade quando construida! Lembre-se também que são sistemas diferentes:

A capacidade de gerar conteúdo tem uma confiugração e sistema exclusivo e individual, com o objetivo de gerar conteúdo dos respectivos campos!

O modal de Editar atividades tem o objetivo de armazenar os campos com todos os conteúdos de dentro de cada respectivo campo, e servir como o construtor de atividades próprio e isolado!

A capacidade de criar atividades, tem o objetivo de ser um sistema de integração para ativação de funcionalidades e chamadas já existentes dentro do sistemas, nesse caso para a ativação direta do botão de Gerar atividades de dentro de cada um dos modais de Editar atividades, de dentro de cada uma das atividades de dentro do card de Interface de Construção!

Os sistemas não precisam trabalhar juntos dentro de um mesmo flow continuo de ativação, mas precisam trabalhar muito bem de maneira independente, pois cada etapa completa a outra!

- Mostrar instrução "Comece a buscar pessoas, digite um nome ou @username para encontrar pessoas para conectar." (p, cinza claro #B0B0B0, italic, font-size 12px).

4. Clica "Seguir perfil", modal atualiza com alerta de sucesso.

**Considerações**:
- Use CSS Flexbox para alinhar elementos no li (ex.: foto à esquerda, texto e botões à direita).
- Adicione debounce (200ms) no input para evitar buscas excessivas (use lodash.debounce).
- Teste responsividade em telas menores (ex.: input width 100% em mobile).
- Use uma imagem padrão (ex.: placeholder.png) se avatar_url for null.

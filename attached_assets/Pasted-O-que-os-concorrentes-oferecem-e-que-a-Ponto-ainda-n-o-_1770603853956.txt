Como os grandes agentes de IA entregam resultados
Analisei como Manus 1.6 Max, ChatGPT Canvas e Claude Artifacts funcionam:

Artefatos estruturados — Toda resposta complexa vira um "artefato" com título, tipo, seções parseáveis e visualização própria
Detecção inteligente — O agente detecta automaticamente o que o usuário precisa e escolhe o formato correto
Pré-visualização rica — O artefato tem UI própria com ícone, cor, seções colapsáveis, exportação
Autonomia total — O agente decide a estrutura do documento, não o usuário
A grande ideia: "Atividades em Versão Texto" via Artefatos
Sua ideia é genial porque:

Escala instantânea — Adicionar 30+ tipos novos de atividades sem desenvolver UI interativa para cada
Custo zero de frontend — Reutiliza o sistema de artefatos que já funciona
Qualidade profissional — Cada tipo tem template otimizado com estrutura pedagógica
Vantagem competitiva — O professor pede e recebe, mesmo que seja algo incomum
Como implementar
A ideia é criar um registro de "Atividades Textuais" — templates especializados que geram documentos formatados usando o sistema criar_arquivo que já existe. Cada template teria:

Tipo identificável (ex: prova_personalizada, caca_palavras, rubrica_avaliacao)
Prompt otimizado para gerar conteúdo pedagógico real, não genérico
Ícone e cor únicos para identificação visual
Detecção automática pelo Jota quando o professor pede
O sistema usaria o documento_livre como "motor", mas com prompts especializados para cada tipo de atividade textual, garantindo qualidade e consistência.
